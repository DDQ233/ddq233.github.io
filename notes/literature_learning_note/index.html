1. 看文献的5个认识

  • 看文献是科研的基本功
  • 变被动为主动
  • 小处着手，放眼全部
  • 多多交流
  • 不要迷信文献

2. 文献检索存在的问题

  • 不清楚不同的文献该采用怎样的检索系统
  • 对数据库用法不了解，不太会检索
  • 检索结果不理想，太多或太少
  • 无法得到全文，也不知道如何去下载
  • 对最新的文献不关注，或者不知道如何跟踪

3. 文献检索常用方法

  • 常规法： 利用检索工具查找文献，即以主题、分类、作者等途径，通过检索工具查找

  • 浏览法： 本专业或本学科相关的核心期刊，比如 Nature，每出一期就浏览阅读

  • 追溯法： 查看一篇参考价值大的文献之后，以后面附的参考文献为线索查找文献


4. 文献筛选的要点

  • 重点关注：最新文献、核心文献
  • 英文为主，中文为辅
  • 读标题摘要进行初筛，能不下载就不下载
  • 结合全文进一步筛选
  • 按重要性进行分级，定期回顾、清理

5. 文献阅读数量

阅读数量不是绝对的，但是量变确实可以质变

重视文章的质量，分层次阅读


6. 文献类型

中文综述 → 中文博士/硕士论文 → 英文综述 → 英文研究型文章


7. 文献阅读的方法论

  • 围绕科研课题阅读文献，围绕科研课题寻找相关知识点，形成相关的知识体系

  • 目标为导向

    • 核心论文
    • 国内外该领域明星作者
    • 开宗立派
    • 综述最全工作
    • 中文 - 英文 - 参考文献 - 参考文献

  • 不同阅读目的有不同的阅读方法

    • 回顾重要内容：摘要、结论、讨论
    • 扩充知识面：只读 Introduction
    • 写文章：讨论、摘抄句型
    • 快速进入一个研究领域：精度全文

8. 一篇具体文章如何阅读

  1. 标题：决定是否继续看
  2. 摘要：是否继续看
  3. 通读全文：了解概况
  4. 图、表
  5. 结论、讨论

9. 阅读文献的方法举例

方法一：

  • 先读摘要，然后浏览前言，略读全文，查看图表

10. 如何阅读一篇文献

  • 确定目标：中文了解科普知识，英文了解最新进展。重要的文章精度，其他速读

  • 先综述后论文：看综述该请概念与前沿，看论文掌握思路与方法。综述决定你看多远，论文决定你走多远

  • 分清文献类别：研究方向、研究领域、研究课题。不同类别不同阅读重点

  • 分清文献形式：期刊、专著、网页、PPT、教科书。不同形式不同阅读方式

  • 结合自己实际：查漏补缺，量力而行，知己知彼，厚积薄发

  • 保持质疑：看实验证据的强弱，有自己独立的判断。永远保持怀疑

    • 发现矛盾
    • 找出异同
    • 思考解决

11. 如何速读一篇文献

  • 快速浏览标题
  • 快速扫一眼摘要，明确本文重点
  • 从结论中进一步确定

12. 精读文献

  • 批判性阅读
  • 仔细阅读
  • 一边阅读一边做笔记
  • 与其他文章对照

13. 如何精读一篇文献

  • 阅读前言，了解背景
  • 实验材料与方法
  • 看懂图表
  • 分析与讨论
  • 结论

14. 如何给文献做笔记

14.1 为什么要做笔记？

  • 不容易遗忘重要内容
  • 对第一次精读有帮助
  • 第二次回顾内容时快速定位

14.2 哪些文献需要做笔记？

  • 本领域经典文献
  • 本领域牛人或课题组的文献
  • 本领域代表性学者的文献
  • 其他核心文献
  • 最新文献

14.3 哪些内容需要重点标记？

  • 你有疑问或者时质疑的地方
  • 作者提出的重要观点
  • 你觉得有问题的数据
  • 你想学习的词汇或句式

  • 基本信息：题目、出处、作者、期卷
  • 中心思想：50 个字的概括目的、手段、结论
  • 实验方案：方案、框架
  • 专题进展：内容汇总后分析
  • 英文句式：单词、引言/讨论怎么写

14.4 如何做笔记

  • 划底线
  • 底线外再加一条线
  • 空白处编号
  • 空白处记下其他页码
  • 关键词圈出来
  • 空白处做笔记
  • 空白处加符号
  • 不同颜色区分
