网页元信息定义

Date : 2020-04-14

使用 <meta> 标签可以定义网页的元信息。

<meta> 标签位于文档的头部，<head> 标签内不包含任何内容。使用 <meta> 标签的属性可以定义与文档相关联的名称/值对。


<meta>标签属性表

属性 说明
content 必须的，定义与 http-equiv 或 name 属性相关联的元信息
http-equiv 把 content 属性关联到HTTP头部，取值包括：content-type、expires、refresh、set-cookie
name 把 content 关联到一个名称，取值包括：author、description、keywords、generator、revised 等
scheme 定义用于翻译 content 属性值的格式
charset 定义文档的字符编码

示例代码：

<!DOCTYPE html>
<html>
    <head>
        <!-- 设置编码 -->
        <meta http-equiv="content-type" content="text/html;charset=utf-8" />
        <!-- 设置页面语言 -->
        <meta http-equiv="content-language" content="zh-CN" />
        <!-- 设置页面刷新时间或跳转页面 -->
        <meta http-equiv="refresh" content="5; url=https://cn.bing.com" />
        <!-- 设置网页缓存时间 -->
        <meta http-equiv="expires" content="Tuesday 14 April 2020 8:00 GMT"/>
        <!--  设置网页不缓存 -->
        <meta http-equiv="pragma" content="no-cache"/>
        <!-- 设置网页关键字 -->
        <meta name="keywords" content="HTML" />
        <!-- 设置网页描述 -->
        <meta name="description" content="description" />
        <!-- 设置网页作者 -->
        <meta name="author" content="DDQ"/>
        <!-- 设置网页版权 -->
        <meta name="copyright" content="https://ddq233.github.io/"/>
        <!-- 设置创建时间 -->
        <meta name="date" content="2020-04-14T08:00:00+00:00"/>
        <!-- 设置禁止搜索引擎检索 -->
        <meta name="robots" content="none"/>

        <title><meta>标签属性</title>
    </head>
<body>
    <p>初识 HTML</p>
</body>
</html>
