CentOS 7 搭建 Nextcloud

Date : 2020-04-07

一、环境准备

  • CentOS 7.5 64位
  • LNMP
    • Nginx - 1.16.0
    • MySQL - 5.6.44
    • Pure-Ftpd - 1.0.49
    • PHP - 7.2
    • phpMyAdmin - 4.7
  • Nextcloud - 17.0.0

注意事项：

  • MySQL 5.5 建议安装内存小于 1GB
  • MySQL 5.6 建议安装内存大于 1GB
  • MySQL 5.7 建议安装内存大于 2GB

二、安装步骤


1. 在 CentOS 安装宝塔面板

一键安装脚本（在 root 用户下执行命令）

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

宝塔面板安装完成


可以在 CentOS 查看宝塔面板

命令：

# bt


2. 进入宝塔面板配置网站环境

在浏览器访问：http://111.229.163.109:8888/1d7bc0e4

输入上面的用户名与密码进行登录


进入宝塔面板后选择以上的 LNMP 环境的套件版本进行安装


等待安装完成，完成后消息盒子任务列表会清空


3. 配置网站

点击：网站 —> 添加站点

如果没有域名则填服务器IP地址


4. 安装 Nextcloud

进入网站目录，删除 /www/wwwroot/nextcloud/ 目录下的四个文件


下载 Nextcloud 17.0.0 压缩文件并上传


上传完成后进行解压，解压后可能会覆盖 nextcloud 目录，因为压缩文件自带了一个 nextcloud 目录


5. 网盘配置初始化

如有域名，在浏览器直接访问域名，如果没有，则访问浏览器的IP


信息确认无误后，点击安装完成，等待数分钟后自动进入网盘界面了
